- ### 가상환경(virtual environment)   
   
   - #### 가상환경 설치
   ```
   # apt-get install python3-venv
   ```
   
   - #### 가상환경 생성
   ```
   # python3 -m venv myvenv (myvenv는 생성하려는 가상환경의 명칭)
   ```
   
   - #### 가상환경 실행하기
   * 실행 시 주의할 점 가상환경파일이 생성된 곳에서 커맨드를 실행해야한다.
   ```
   # source myvenv/bin/activate
   ```
   
   - #### 가상환경 나가기
   ```
   # deactivate
   ```
